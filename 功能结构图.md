```
┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐

┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
│ 交易    │◀───│ 确认    │◀───│ 卖家    │◀───│ 支付    │◀───│ 支付    │

## 数据库E-R图概念模型

```
┌─────────────┐       ┌─────────────┐       ┌─────────────┐
│   用户表    │       │   商品表    │       │   订单表    │
│   (user)   │       │  (goods)   │       │(trade_order)│
├─────────────┤       ├─────────────┤       ├─────────────┤
│ id (PK)    │───┐   │ id (PK)    │       │ id (PK)    │
│ openid     │   │   │ user_id(FK)│───┐   │ goods_id(FK)│───┐
│ nickname   │   └──▶│ buyer_id   │   │   │ buyer_id   │   │
│ avatar_url │       │ seller_id  │   │   │ seller_id  │   │
│ phone      │       │ title      │   │   │ order_no   │   │
│ status     │       │ price      │   │   │ amount     │   │
└─────────────┘       │ status     │   │   │ order_status   │
                      │ category_id│   │   │ pay_status │   │
                      │ created_at │   │   │ created_at │   │
                      └─────────────┘   │   └─────────────┘   │
                               │        │           │        │
                               │        │           │        │
                      ┌────────┴┐      │    ┌──────┴───────┐ │
                      │ 分类表  │      │    │  商品收藏表  │ │
                      │(category)      │    │(goods_favorite)│
                      ├─────────┤      │    ├─────────────┤ │
                      │ id (PK) │      │    │ id (PK)     │ │
                      │ name    │      │    │ goods_id(FK)│─┘
                      │ sort    │      │    │ user_id(FK) │◀─┘
                      └─────────┘      │    └─────────────┘

                      ┌────────────────┴┐
                      │  商品留言表     │
                      │(goods_comment) │
                      ├────────────────┤
                      │ id (PK)       │
                      │ goods_id (FK) │
                      │ user_id (FK)  │
                      │ content       │
                      │ created_at    │
